# 卷积神经网络（CNN）

## 是什么

专门用来处理图片的神经网络。

## 为什么不用全连接层

```
一张图片100万个像素
全连接层：每个神经元看全部100万像素 → 参数太多，训练不动
CNN：    每个神经元只看一小块（3×3）→ 参数很少，能训练
```

CNN利用图片的两个特性：
- 平移不变性：猪在左上角还是右下角，都是猪
- 局部性：看局部就够了，不用看全图

## CNN的三个核心组件

### 1. 卷积层（提取特征）

用一个小方块（卷积核，比如3×3）在图片上滑动，每次看一小块区域，提取边缘、纹理等特征。

卷积核 = 可学习的参数，通过训练学出来。

### 2. 池化层（压缩信息）

把图片变小，保留重要信息。

最大池化：在2×2方块里只留最大值。
28×28 → 池化后 → 14×14

池化没有参数，是固定操作。

### 3. 全连接层（做决定）

卷积和池化做完后，综合所有特征判断这是什么。

## LeNet结构（第一个CNN模型）

```
图片 → 卷积 → 池化 → 卷积 → 池化 → 全连接 → 全连接 → 答案
       提取特征  压缩   提取特征  压缩    判断      判断
```

具体：
```
输入(1×28×28)
  → 卷积(5×5, 6个通道) + Sigmoid + 池化(2×2)
  → 卷积(5×5, 16个通道) + Sigmoid + 池化(2×2)
  → 展平
  → 全连接(120) → 全连接(84) → 全连接(10)
```

通道数变化：1 → 6 → 16
尺寸变化：28×28 → 14×14 → 5×5

## 关键概念

| 概念 | 说明 |
|------|------|
| 卷积核 | 小方块（3×3），在图片上滑动提取特征 |
| 填充(Padding) | 在图片边缘填0，防止图片越来越小 |
| 步幅(Stride) | 卷积核每次跳着走，让图片快速变小 |
| 通道 | 多个卷积核 → 多个输出 → 提取多种特征 |
| 感受野 | 网络越深，每个神经元看到的区域越大 |

## 一句话总结

CNN = 用小放大镜看局部（卷积）+ 压缩信息（池化）+ 最终判断（全连接）
